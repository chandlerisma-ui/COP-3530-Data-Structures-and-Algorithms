# Data Structures & Algorithms — FAU COP 3530

A collection of assignments from the Data Structures & Algorithms course at
Florida Atlantic University. Projects focus on implementing and analyzing
fundamental data structures and algorithms in C++.

## Topics Covered
- Linked Lists (singly, doubly, circular)
- Trees & Graphs (binary trees, BSTs, traversals)
- Stacks & Queues
- Hash Tables
- Sorting & Searching (merge sort, quick sort, binary search, etc.)
- Dynamic Programming

## Languages
- C++
- C
- Python
- Psuedocode
